# Notes APP

Markdown notes app built with React ✍️

## Install

```bash
git clone https://github.com/balewgize/notes-app
```

```bash
cd notes-app
```
```bash
npm install
```

## Usage

```bash
npm run dev
```

### Built with
- React
- Vite
- react-mde
- react-split
- nanoid
- showdown
